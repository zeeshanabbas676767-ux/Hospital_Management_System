# Hospital Management System

Hospital Management System is a web application built using **ASP.NET MVC (Framework)** that users can know about how system working in every hospital.  
It also includes **user authentication (login/signup in this project password hide and show feature is added)**, profile management, and an admin panel for managing content.

---

## ✨ Features

- 🪐 Browse hospital managements topics (Appointment, Hospitals, Departments, Doctors, Patients, Invoice, User)
- 👥 User authentication (Login, Signup, Logout)
- 🖼️ Profile pictures and user settings
- 📱 Responsive UI with light and dark theme
- 🧩 CRUD operations on data
---

## 💻 Tech Stack

| Technology | Description |
|-------------|-------------|
| **ASP.NET MVC 5** | Web application framework |
| **C#** | Backend programming language |
| **Entity Framework** | ORM for database operations |
| **SQL Server** | Database |
| **HTML5 / CSS3 / Bootstrap** | Frontend and styling |
| **JavaScript / jQuery** | Dynamic UI features |
| **Visual Studio 2022** | IDE used for development |
